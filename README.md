# Buchtausch-App Database

## Projektbeschreibung
Dieses Projekt beschreibt die Entwicklung einer relationalen Datenbank für eine Buchtausch-App. Benutzer:innen können Bücher anbieten, ausleihen, bewerten und Nachrichten austauschen.

Die Datenbank wurde mit MariaDB/MySQL implementiert und enthält verschiedene Tabellen, Beziehungen, Constraints und Testfälle.

---

## Verwendete Technologien
- MariaDB / MySQL
- SQL
- XAMPP
- GitHub
- PowerPoint

---

## Funktionen der Datenbank
- Verwaltung von Benutzer:innen
- Verwaltung von Büchern
- Ausleihsystem
- Bewertungen und Kommentare
- Nachrichten zwischen Benutzer:innen
- Kategorien und Standorte
- Geografische Suche
- Testfälle und JOIN-Abfragen
- Datenintegrität mit Constraints
- Optimierung mit Indizes

---

## Datenbankstruktur
Die Datenbank besteht aus mehreren relationalen Tabellen:
- benutzer
- buch
- verfuegbarkeit
- ausleihe
- bewertung
- kategorie
- buch_kategorie
- nachricht
- standort
- buch_standort

---

## Installation
1. XAMPP starten
2. MariaDB/MySQL aktivieren
3. SQL-Datei importieren
4. Datenbank „buchtausch_app“ verwenden
